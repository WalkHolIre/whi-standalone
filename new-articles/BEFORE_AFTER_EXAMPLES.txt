Issues:

- Inconsistent field naming
- Description was ~180 chars (too long)
- Missing /blog/ prefix on slug
- Date field not needed in standard format

-----
---

Improvements:
- Consolidated to 4 standard fields
- Description trimmed to 155 characters
- Added /blog/ prefix to slug
- Removed redundant metadata fields
- Clean, consistent YAML structure

All 29 articles now follow this consistent format:
